### Inspiration

A personal macropad to easily navigate my notes when drawing with my Wacom.


### Specifications

BOM: 
- 4x Cherry MX Switches
- 4x SK6812 MINI Leds
- 1x XIAO RP2040
- 4x Blank DSA Keycaps
- 4x M3x16 Bolt
- 4x M3 Heatset
- 1x 0.91" 128x32 OLED Display
